# Sistema de controle financeiro
## Tecnologia utilizada:
- React
- Typescript
- MirageJS
- Styled-Components
- Context API
- Axios

## Descrição
Esse sistema foi desenvolvido junto ao bootcamp da Rocketseat Ignite e tem como objetivo aplicar ferramentas conceitos mais utilizados no mercado.
